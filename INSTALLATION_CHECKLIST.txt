   https://nodejs.org
   
2. npm (kommt mit Node.js)
   
3. Next.js 14
   npm install next
   
4. React 18
   npm install react react-dom
   
5. Tailwind CSS
   npm install tailwindcss
   
6. React Hook Form
   npm install react-hook-form
   
7. Nodemailer (für Email)
   npm install nodemailer
   
8. Google Analytics
   - Account: https://analytics.google.com
   - Measurement ID in .env.local eintragen
   
9. Usercentrics (Cookie Consent)
   - Account: https://usercentrics.com
   - Script zu Footer hinzufügen
   
10. STRATO Hosting (aktueller Host)
    - Domain: impub.de
    - Mail: info@impub.de
